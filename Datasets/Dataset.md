**Usage of Dataset**

The dataset is used to train and evaluate an AI-based pest classification system that identifies rice pests from crop images and supports intelligent treatment and prevention recommendations. It enables robust learning of visual pest features under real-world farming conditions.
Dataset Information



- Dataset Name: IP102 Pest Dataset
-  Source: Public Research Dataset (Wu et al., 2019)
-  Domain: Agriculture / Computer Vision
-  Task: Multi-class Image Classification
-  Problem Type: Supervised Learning
-  File Format: JPG / PNG images with label mapping (.txt)
-  Dataset Link: **[Download Dataset](https://drive.google.com/drive/folders/157UriEEQAsYMoC7rBr_q0I33h5qDWG-3?usp=sharing)**


**Dataset Overview**
- Total Records: ~75,000+ images
- Labeled Records: All images are labeled
- Classes: 40 rice pest classes (subset used)
- Annotation Type: Image-level class labels


**Why This Dataset?**

- Contains real-world pest images with diverse lighting and backgrounds
- Includes visually similar pest species, making classification challenging
- Widely accepted benchmark dataset in agricultural AI research
- Suitable for evaluating deep learning and transformer-based models


**Features Used**

- Feature 1: Pest visual patterns (shape, texture, color)
- Feature 2: Leaf and crop context information
- Feature 3: Spatial and structural features captured via image patches


**Summary**

The IP102 dataset provides a comprehensive and challenging foundation for building high-accuracy pest classification models. Its diversity, scale, and real-world relevance make it ideal for developing and validating intelligent agricultural decision-support systems.
